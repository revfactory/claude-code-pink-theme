## 정보 구조 (IA) 원칙

1. **점진적 공개**: 초보자부터 고급 사용자까지 단계별로 정보 제공
2. **작업 중심**: 사용자가 달성하려는 목표별로 콘텐츠 구성
3. **크로스 링킹**: 관련 콘텐츠 간 연결로 탐색 용이성 향상
4. **검색 최적화**: 명확한 제목과 메타데이터로 찾기 쉽게 구성
