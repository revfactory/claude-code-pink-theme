시스템 요구사항

운영체제

✅ 지원되는 운영체제

  • macOS: 10.15 (Catalina) 이상
  • Linux: Ubuntu 20.04+, Debian 10+
  • Windows: WSL (Windows Subsystem for Linux) 사용

하드웨어 요구사항

💾 메모리

최소: 4GB RAM

권장: 8GB RAM 이상

💿 저장공간

최소 500MB의 여유 공간 필요

🌐 네트워크

안정적인 인터넷 연결 필수

소프트웨어 요구사항

📦 Node.js

버전: 18.0 이상 (필수)

Node.js가 설치되어 있지 않다면 nodejs.org에서 다운로드하세요.

🌿 Git (선택사항)

버전: 2.23 이상

Git 관련 기능을 사용하려면 설치가 필요합니다.

🔍 ripgrep (선택사항)

빠른 검색 기능을 위해 설치를 권장합니다.

🛠️ GitHub/GitLab CLI (선택사항)

PR 생성 등 고급 Git 기능을 사용하려면 설치가 필요합니다.

지역 제한

🌍 사용 가능 지역

Claude Code는 Anthropic API가 지원되는 국가에서만 사용할 수 있습니다. 자세한 지역 목록은 공식 문서를 참조하세요.

버전 확인 방법

# Node.js 버전 확인
node --version
v18.17.0

# Git 버전 확인
git --version
git version 2.39.0

# npm 버전 확인
npm --version
9.6.7

문제 해결

시스템 요구사항과 관련된 문제가 발생하면:

  1. 모든 필수 소프트웨어가 올바른 버전으로 설치되어 있는지 확인하세요
  2. 운영체제가 지원되는 버전인지 확인하세요
  3. 인터넷 연결이 안정적인지 확인하세요
  4. 문제가 지속되면 문제 해결 가이드를 참조하세요
