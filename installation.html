설치 가이드

빠른 설치

Claude Code를 설치하는 가장 빠른 방법입니다:

# npm을 사용한 전역 설치
npm install -g @anthropic-ai/claude-code

⚠️ 중요한 주의사항

  • sudo를 사용하지 마세요! 권한 문제가 발생할 수 있습니다.
  • Node.js 18 이상이 설치되어 있어야 합니다.
  • 안정적인 인터넷 연결이 필요합니다.

단계별 설치 과정

  1. Node.js 설치 확인 
    node --version
v18.17.0
  2. Claude Code 설치 
    npm install -g @anthropic-ai/claude-code
  3. 프로젝트 디렉토리로 이동 
    cd your-project
  4. Claude Code 시작 
    claude

인증 설정

Claude Code를 처음 실행하면 인증이 필요합니다. 다음 중 하나를 선택할 수 있습니다:

🔑 Anthropic Console (기본)

웹 브라우저가 자동으로 열리며 Anthropic Console에서 인증을 완료합니다.

💳 Claude App (Pro/Max 플랜)

Claude Pro 또는 Max 구독자는 Claude 앱을 통해 인증할 수 있습니다.

🏢 엔터프라이즈 플랫폼

Amazon Bedrock 또는 Google Vertex AI를 통한 인증도 지원됩니다.

첫 사용자를 위한 권장사항

🎯 추천 첫 단계

  1. Claude Code를 시작합니다: claude
  2. 간단한 명령어로 테스트: summarize this project
  3. 프로젝트 가이드 생성: /init
  4. 생성된 CLAUDE.md 파일을 커밋합니다

플랫폼별 설치 가이드

macOS

# Homebrew로 Node.js 설치 (필요한 경우)
brew install node

# Claude Code 설치
npm install -g @anthropic-ai/claude-code

Linux (Ubuntu/Debian)

# Node.js 설치 (필요한 경우)
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# Claude Code 설치
npm install -g @anthropic-ai/claude-code

Windows (WSL)

# WSL 터미널에서 실행
# Node.js 설치 (필요한 경우)
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# Claude Code 설치
npm install -g @anthropic-ai/claude-code

설치 확인

설치가 완료되었는지 확인하려면:

claude --version
@anthropic-ai/claude-code v1.0.0

문제 해결

설치 중 문제가 발생했나요? 문제 해결 가이드를 참조하거나 다음을 시도해보세요:

  • npm 캐시 정리: npm cache clean --force
  • Node.js 버전 확인: node --version
  • 권한 문제 해결: npm 전역 디렉토리 권한 확인
